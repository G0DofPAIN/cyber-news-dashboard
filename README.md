# Cyber News Dashboard

Static cybersecurity news dashboard generated by a Python script.

## Files

- `index.html` — the page served by GitHub Pages.
- `fetch_cyber_news.py` — fetches RSS/HTML sources and rebuilds `index.html`.
- `cyber_news_history.json` — local history/cache of collected items.

## Run locally

```bash
python fetch_cyber_news.py
```

Then open:

```text
index.html
```

## Publish with GitHub Pages

1. Create a GitHub repository, for example `cyber-news-dashboard`.
2. Upload/commit all files from this package.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`, then Save.

Your site will be available as:

```text
https://USERNAME.github.io/cyber-news-dashboard/
```

## Automatic updates

The included GitHub Actions workflow runs the Python script every 6 hours and commits the updated `index.html` and `cyber_news_history.json` back to the repository.

If GitHub Actions asks for permissions, set:

**Settings → Actions → General → Workflow permissions → Read and write permissions**.
